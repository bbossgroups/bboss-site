# bboss官方网站及bboss demo工程
 

# 工程构建运行说明：
## 1.搭建好gradle构建环境及gradle环境变量

参考文档：http://yin-bp.iteye.com/blog/2313145


## 2.运行工程根目录下的build.bat指令，或者手动在命令行下运行gradle install指令
## 3.构建成功后,会生成war包：


build/libs/bboss-site-xxxx.war
